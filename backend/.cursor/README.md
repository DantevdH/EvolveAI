# Backend Cursor Rules

This directory contains backend-specific rules for the Cursor AI assistant.

## Rules

- **python.md** - Python code style, type hints, and best practices
- **testing.md** - Backend testing guidelines, pytest setup, mocking
- **fastapi.md** - FastAPI API design, routes, and patterns

## Scope

These rules apply specifically to the backend codebase:
- Python code in `backend/`
- FastAPI routes and endpoints
- Backend testing with pytest
- Database operations with Supabase
- Environment variable management
