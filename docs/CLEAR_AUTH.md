# Clear Authentication

Quick way to clear auth tokens and redirect to login page for testing.

## How to Use

1. Open React Native DevTools Console
2. Type in the console:
   ```javascript
   clearAuth()
   ```
3. Press Enter

This will:
- Clear all authentication tokens
- Sign out from Supabase
- Redirect to login page

**Note:** Only available in development mode (`__DEV__`).
