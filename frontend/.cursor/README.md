# Frontend Cursor Rules

This directory contains frontend-specific rules for the Cursor AI assistant.

## Rules

- **typescript.md** - TypeScript code style and best practices
- **react-native.md** - React Native patterns, components, and performance
- **testing.md** - Frontend testing guidelines with Jest and React Native Testing Library

## Scope

These rules apply specifically to the frontend codebase:
- TypeScript/JavaScript code in `frontend/`
- React Native components and screens
- Expo Router navigation
- Frontend testing with Jest
- TypeScript types and interfaces
